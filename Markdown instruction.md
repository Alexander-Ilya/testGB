* Элемент 1
* Элемент 2
* Элемент 3
* Элемент 4
* Элемент 5

чтобы добавить нумерованные списки, необходимо пункты просто пронумеровать. Например, вот так
1. Элемент 1
2. Элемент 2
3. Элемент 3
4. Элемент 4
5. Элемент 5
